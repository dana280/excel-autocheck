## 🎯 איך להשתמש?

### מצב 1: מטלה בודדת (כמו קודם)
1. בחרי "מטלה בודדת"
2. העלי מחוון
3. העלי מטלת תלמיד
4. "בצע בדיקה"

### מצב 2: 50 מטלות בבת אחת! 🚀
1. **בחרי "מספר מטלות (Batch)"**
2. **העלי קובץ מחוון אחד**
3. **העלי את כל 50 קבצי המטלות** (Ctrl+Click לבחירה מרובה)
4. (אופציונלי) הזיני מזהי תלמידים
5. **"בצע בדיקה לכל המטלות"**
6. המתיני 2-5 דקות ⏱️

### מה תקבלי?

#### 📁 לכל מטלה:
קובץ המטלה המקורי + **גליון חדש בשם "🎓 גליון_בדיקה"**

הגליון כולל:
- ✅ ציון כולל (גדול וצבעוני)
- ✅ טבלה מפורטת עם כל הבדיקות
- ✅ צבעים: 🟢 ירוק (עבר), 🟡 כתום (חלקי), 🔴 אדום (נכשל)
- ✅ הערות מפורטות על כל בדיקה

#### 📊 קובץ סיכום Excel:
טבלה אחת עם **כל המטלות**:
- מספר מטלה
- ציון
- אחוז
- סטטוס
- **הערות על מה ירד!** ⬅️ זה מה שרצית!

דוגמה:
```
| מטלה | ציון | אחוז | הערות_מה_ירד                    |
|------|------|------|----------------------------------|
| 001  | 85   | 85%  | נוסחאות: -5 נקודות              |
| 002  | 92   | 92%  | הכל תקין                        |
| 003  | 68   | 68%  | תנאים: -10, חישובים: -12 נקודות|
```

#### 📦 ארכיון ZIP:
כל הקבצים ביחד - **לחצי כפתור אחד והכל מוכן להורדה!**

## 🤖 שימוש ב-Claude API (אופציונלי אבל מומלץ!)

### למה זה טוב?
- 🧠 בדיקות חכמות יותר
- 🔍 זיהוי טעויות לוגיות
- 💡 המלצות לשיפור
- ✨ ניתוח מורכב של נוסחאות

### איך להגדיר?

#### שלב 1: קבלת API Key (יש תקופת ניסיון חינמית!)
1. גשי ל-https://console.anthropic.com
2. צרי חשבון
3. "API Keys" → "Create Key"
4. העתיקי את המפתח (מתחיל ב-`sk-ant-...`)

#### שלב 2: הגדרה בממשק
1. בסרגל הצד: ✅ "השתמש ב-Claude API"
2. הדביקי את המפתח
3. זהו!

#### שלב 3: הגדרה ב-Streamlit Cloud (לפרודקשן)
1. Streamlit Cloud → האפליקציה → "Settings" → "Secrets"
2. הוסיפי:
```toml
ANTHROPIC_API_KEY = "sk-ant-api03-..."
```
3. Save
