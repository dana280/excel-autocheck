# ⚡ התחלה מהירה - מערכת בדיקת מטלות אקסל

## 🎯 מה קיבלת?

מערכת שלמה לבדיקת מטלות אקסל אוטומטית עם ממשק web ידידותי!

---

## 📦 הקבצים שלך

### קבצי הקוד העיקריים:
1. **`streamlit_app.py`** - אפליקציית Web (הממשק הגרפי)
2. **`excel_checker_advanced.py`** - מנוע הבדיקה (הלוגיקה)
3. **`requirements.txt`** - רשימת חבילות נדרשות

### קבצי תיעוד:
4. **`README.md`** - תיעוד מלא של המערכת
5. **`DEPLOYMENT_GUIDE.md`** - מדריך פריסה מפורט
6. **`LICENSE`** - רישיון MIT

### קבצי הגדרות:
7. **`.gitignore`** - קבצים להתעלמות ב-Git
8. **`.streamlit/config.toml`** - הגדרות Streamlit

### דוגמאות:
9. **`examples/`** - תיקייה עם קבצי דוגמה

---

## 🚀 3 דקות להתחלה

### אופציה 1: הרצה מקומית (לבדיקה)

```bash
# 1. התקן חבילות
pip install -r requirements.txt

# 2. הרץ את האפליקציה
streamlit run streamlit_app.py

# 3. פתח דפדפן: http://localhost:8501
```

### אופציה 2: פרסום ב-Streamlit Cloud (מומלץ!)

#### שלב א': העלה ל-GitHub

1. צור חשבון ב-[GitHub.com](https://github.com)
2. צור repository חדש בשם `excel-checker`
3. העלה את כל הקבצים (גרור ושחרר)

#### שלב ב': פרסם ב-Streamlit Cloud

1. גש ל-[share.streamlit.io](https://share.streamlit.io)
2. התחבר עם GitHub
3. לחץ "New app"
4. בחר:
   - Repository: `your-username/excel-checker`
   - Branch: `main`
   - Main file: `streamlit_app.py`
5. לחץ "Deploy"
6. חכה 2-3 דקות ✨

**זהו! קיבלת URL לשיתוף עם הצוות! 🎉**
