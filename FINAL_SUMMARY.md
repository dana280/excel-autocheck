### 1. "שאוכל להוריד אליו כ50 מטלות"
✅ **תשובה**: יש! הוספתי מצב **Batch** שמטפל ב-50 מטלות בבת אחת.

### 2. "שאקבל את המטלה של הסטודנט עם גליון נוסף בתוך המטלה"
✅ **תשובה**: כן! המערכת מוסיפה **גליון בשם "🎓 גליון_בדיקה"** לכל קובץ מטלה.

- ציון כולל (גדול וצבעוני)

- הערות מפורטות

### 3. "לפי המחוון ומה ירד כולל ציון סופי"

- ציון סופי
- פירוט של כל סעיף מהמחוון
- **בדיוק מה ירד וכמה נקודות**

### 4. "וכן קובץ אקסל עם מספר המטלה, הערות על מה ירד וציון"

| מטלה_1    | 85   | 100     | 85%  | נוסחאות: -5 נקודות, תנאים: -10 נקודות |
| מטלה_2    | 92   | 100     | 92%  | הכל תקין |

## 🚀 תרחיש שימוש מלא - 40 מטלות

### 📁 **לפני:**
יש לך:
- 1 קובץ מחוון
- 40 קבצי מטלות מתלמידים

### 💻 **תהליך:**

#### שלב 1: פתיחת המערכת
```
https://excel-checker-yourname.streamlit.app
```

#### שלב 2: בחירת מצב
בחרי: **"מספר מטלות (Batch)"**

#### שלב 3: העלאת קבצים
1. העלי את **קובץ המחוון** (1 קובץ)
2. העלי את **כל 40 המטלות** (Ctrl+Click לבחירה מרובה)

#### שלב 4: הרצת בדיקה
לחצי: **"בצע בדיקה לכל המטלות"**
המתיני: 3-4 דקות ⏱️

### 📥 **אחרי:**

תקבלי 3 דברים:

#### 1️⃣ **40 קבצים עם גליון בדיקה**
כל קובץ מטלה + גליון חדש:

```
קובץ: מטלה_תלמיד_1.xlsx
├─ Sheet1 (המטלה המקורית)
├─ Sheet2 (המטלה המקורית)
└─ 🎓 גליון_בדיקה ⬅️ חדש!
   ├─ ציון כולל: 85/100 (85%)
   ├─ טבלה:
   │  ├─ חישובים: ✅ עבר (10/10)
   │  ├─ נוסחאות: ⚠️ חלקי (7/10) - חסר SUM
   │  └─ תנאים: ❌ נכשל (0/5) - אין IF
   └─ הערות מפורטות
```

#### 2️⃣ **קובץ סיכום Excel אחד**
```
קובץ: סיכום_כל_המטלות.xlsx
```

| מטלה | ציון | % | הערות_מה_ירד |
|------|------|---|---------------|
| 001  | 85   | 85| נוסחאות: -3, תנאים: -5, חישובים: -7 |
| 002  | 92   | 92| הכל תקין ✅ |
| 003  | 65   | 65| נוסחאות: -10, תנאים: -15, חישובים: -10 |
| ...  | ...  | ..| ... |

עם צבעים:
- 🟢 ירוק: 80%+
- 🟡 לבן: 60-79%
- 🔴 אדום: <60%

#### 3️⃣ **ארכיון ZIP**
כל 40 הקבצים + הסיכום בקובץ אחד להורדה!

## 🤖 תכונת Claude API - למה זה חשוב?

### ללא API (בדיקה רגילה):
✅ בודק אם יש נוסחאות
✅ בודק אם יש פונקציות
✅ בודק אם יש גליונות

### עם API (בדיקה חכמה):
✅ כל מה שלמעלה +
🧠 בודק אם הנוסחאות **נכונות לוגית**
🧠 מזהה טעויות מתמטיות
🧠 מציע שיפורים
🧠 מבין הקשר עסקי

**דוגמה:**
```
ללא API: "יש נוסחת SUM ✅"
עם API: "יש נוסחת SUM אבל היא מחברת טווח שגוי. 
         צריך להוסיף את תאים D5:D10 ⚠️"
```
